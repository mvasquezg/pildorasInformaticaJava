en el ano 1991 Sun MicroSystem creo Java (James Gosling y Patrick Naughton) y Oracle la compro

Antes de llamarse Java quiseorn llamarle Oak, pero este ya estaba registrado.

catacteristicas de Java
-- Orientado a Objetos
-- Distribuido. Buen tratamiento de redes. Buena programacion para internet.
-- Seguro. Como fue pensado para trabajar en intrenet se hizo seguro
    --No permite leer o esribir ficheros sin permiso
    --desbordar la pila de ejecucion
    --Corrupcion de memoria
--Neutro (multiplataforma)
--Adaptable: tipo de datos primitivos iguales en todas las plataformas.
--Interpretado.
--Alto rendimiento.
--Multi treading.

Java no es extension de Html
java no es JavaScript
Todos los programas no se ejecutan en una pagina web.
Java no es inseguro, pero no es 100% seguro.

video 13 (11) empezar 19/Abril/23
Manipulacion de String

String no es dato de tipo primitivo

metodo length devuelve la longitud de una cadena
metodo charAt(n) devuelve la posicion de un caracter dentro de una cadena. (Las posiciones empiezan a contar de Cero)
metodo substring(x, y) devuelve una subcadena dentro de la cadena, siendo X el caracter a partir del cual se extrae e Y el numero de caracteres que se quieren extraer. 
metodo equals(cadena) devuelve true si dos cadenas que se comparan son iguales y false si no lo son. Distingue mayusculas y minusculas.
metodo equalsIgnoreCase igual que el metodo equals pero sin tener en cuenta mayusculas o minusculas.
