Video 5 (3)
en el ano 1991 Sun MicroSystem creo Java (James Gosling y Patrick Naughton) y Oracle la compro
Java se creo para que se pudiera utilizar en pequenos electrodomesticos.

Antes de llamarse Java quiseorn llamarle Oak, pero este ya estaba registrado.

Los ingenieros eran aditctos al cafe, en USA coloquialmente tomar una taza de cafe es iguala java

En el ano 94 e internet se hacia mas grande, pensaron que las catacteristicas de java se adaptaba como guante internet

1996 se creo un navegador llamado hot Java

catacteristicas de Java
-- Orientado a Objetos
-- Distribuido. Buen tratamiento de redes. Buena programacion para internet.
-- Seguro. Como fue pensado para trabajar en intrenet se hizo seguro
    --No permite leer o esribir ficheros sin permiso
    --desbordar la pila de ejecucion
    --Corrupcion de memoria
--Neutro (multiplataforma)
--Adaptable: tipo de datos primitivos iguales en todas las plataformas.
--Interpretado.
--Alto rendimiento.
--Multi treading.

Java no es extension de Html
java no es JavaScript
Todos los programas no se ejecutan en una pagina web.
Java no es inseguro, pero no es 100% seguro.
